### Multimodal Legacy Document Intelligence System (OCR + LLM)
An enterprise-grade document extraction pipeline built to digitize, parse, and structure low-resolution, degraded PDFs containing mixed media (tables, text, graphs, and images) into queryable database entries.

| Metric | Details |
| :--- | :--- |
| **Stack** | Python, OpenCV, TensorFlow, Transformers, NumPy, Pandas, MongoDB, Tesseract OCR, Qwen2-VL |
| **Scale** | Fully automated ingestion and extraction of 1000+ pages of degraded historical documents |
| **Performance** | Table extraction structure alignment improved by 35%; OCR layout segmentation accuracy > 92% |
| **Security** | Local processing pipelines to ensure complete data privacy and no external model leakage |
| **Impact** | Automates administrative data entry backlogs, reducing document processing lifecycle from days to minutes |
| **Repository** | [document-intelligence](https://github.com/Sanyam0385/document-intelligence) |

#### Technical Deep-Dive
* Designed and executed a hybrid layout analysis pipeline using OpenCV for morphological contour analysis and grid-line segmentation, enabling highly accurate table boundary localization.
* Integrated Qwen2-VL (Vision-Language Model) for deep context understanding, capturing complicated multi-line table headers, cell values, and chart legends without manual templates.
* Built a MongoDB schema mapping pipeline converting OCR outputs directly into nested JSON formats ready for semantic search applications.

</details>

| **Performance** | Reduced context misalignment by 40%, cut context drift by 35%, and boosted intent accuracy by 27% |
| **Security** | Sanitized LLM input/output validation layers protecting against prompt injections |
| **Impact** | Seamless conversational experience eliminating latency lags and state drift between audio and text |
| **Repository** | [agentic-dialogue-router](https://github.com/Sanyam0385/agentic-dialogue-router) |

#### Technical Deep-Dive
* Architected a multimodal context-injection dialogue-routing layer that guides LLM responses based on live stream state.
* Developed custom memory-truncation algorithms keeping dialogue context concise and aligned with the exact position of real-time physical TTS audio playbacks.
* Designed an R&D evaluation framework utilizing reinforcement learning principles to grade intent-classification latency.

</details>
